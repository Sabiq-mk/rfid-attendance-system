# 📡 IoT Based Smart Attendance System using RFID

## 📌 Overview

This project is an automated attendance system using RFID and IoT. It replaces manual attendance with a fast, reliable system using NodeMCU and Google Sheets.

## ⚙️ Components Used

* ESP8266 NodeMCU
* RFID Reader (RC522)
* RFID Tags
* Buzzer
* Power Supply

## 🔄 Working

* RFID card is scanned
* Data is sent to NodeMCU
* Attendance is updated in Google Sheets via WiFi

## 📊 Features

* Real-time attendance tracking
* Cloud storage using Google Sheets
* Reduces manual errors

## 🧠 Future Scope

* Mobile app integration
* Biometric authentication
* SMS alerts using GSM module
